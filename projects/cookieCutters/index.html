Rhino Cookie Cutters

This photo is can be found on unsplash here.


I started off by desaturating the image.



From there I adjusted the Levels to try to get the buffalo and background as solid and contrasted as possible.



I used "select color range" in Photoshop to select the black buffalo.



Here I colored it red to help differentiate it from black and stand out more.



I laid the original image with about 30% opacity over the red to help clean up the lines and edges.



For example, I cleaned up the nose slightly to add in more than just the white selected.



I used the magic select tool, and then "expand selection" to get a selection outline around the horns.



This allowed me to remove the blue for the inside of the horn so that I could use this as an edge for my cookie cuttter.



In Illustrator, I used "make and expand" under Image trace, to turn the image into a vector path.



I removed the fill by deleting the path, for some reason I had issues just setting the fill to "none." However I didn't run into this issue with subsequent cookie cutters.



I exported the file to a DXF to import into Rhino.



I imported the file and scaled the image down using "scale"



I used the command "explode" to break this down into different paths.



I accidentally exported the boarder as a vector, but was able to delete it after I used explode.



Here I used "rebuild" to simplify how many points are in the paths.



I used rebuild on each curve since the horns and eyes needed much less vertices.




I used "offset" to create curves for each of the paths of the buffalo so that I could create surfaces using "PlaneSrf" to give the cookie cutter edges a 1/10th of an inch thickness.



From there I was able to select all of the surfaces...



and extrude!



I also created a surface using the outer most curve of the buffalo to create a "top" for the cookie cutter.



While this makes the cutter work for the internal parts, it's not ideal and uses more material. I try out "bridges" in my next cutter!



I used "BooleanUnion" to join all of the meshes before I exported.



Here is the finished buffalo! I exported the object as a Binary STL file.


My next cutter is a mug. You will find the source here.


For this image, I also desaturated, Leveled, and blurred the image to single out the mug.


I inverted the image to work with the black surface.



I cleaned up the mug slightly and removed the caption.



After blurring one more time, I exported to Illustrator.



From here I used the image trace window, and then expanded, in order to get the right tunning.



I went through the same process as the buffalo to offset the curves and create surfaces.



I created another offset curve on the outside to give this cookie cutter a lip, and not use a solid top.



I used "SrfPt" this time to create a surface for a bridge for the interior.



Using the bridge and the lip, I was able to create the top surface that was hollow.



I then extruded the edges of the cutter.



I used BooleanUnion to combine the polysurfaces again, and then exported.


My next cutter is a pine cone! You will find the source here.


I used the magic select tool this time to cut out the pine cone.



I then desaturated, leveled and blurred the pine cone like the previous cutters.



Again a simple image trace and expand.



This cone could use waaaay less verticies.



The pine cone cutter is the simplest, with only two other offset curves for the cutter and the outer lip. I followed the same process of booleanunion and exporting to an STL.
