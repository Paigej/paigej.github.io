Foggy Mountains Flight Simulator

Posted on November 4th, 2016 at 9:45 PM

This is a OpenGL graphics project. I render a foggy mountain terrain based on a height map and overlay the map as a texture.

View the Code on Github


Here you can see the generated mountains without the fog. The mountains are generated inside of a sky box using a height map. The height map was taken from terrain.party from the Wulingyuan Mountain range.



I sample the grayscale pixels on the map on the left to assign a height value. With these values I generate polygons that constitute the mountains. I then overlay the colored heightmap as a texture to give the mountains a purple color that is topped with "snow."



The combination of building the mountains with a real height map and then overlaying a color version is what gave the mountains a more realistic look.



The Fog is generated using OpenGL's fog functions, and is set in the lighting as a Quadratic attenuation fog.



The OpenGL camera is constantly moving forward, giving the user a sense of flying. The direction is controlled with the mouse while the 'w','a','s', and 'd' keys allow for roll, pitch, and yaw.



The user can also use the space bar to fire a "fireball" that is an OpenGL light.



The light travels in the direction the user was looking to light up the scene around it.



The light shows the color of the original texture through the fog, creating a really interesting effect.
